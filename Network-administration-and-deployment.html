Network Administration and Deployment

If you have a previous version of Revit installed on your system, you can install a newer version of Revit and keep other versions of the program on the same system. This is called a side-by-side installation. If you have purchased an upgrade version of Revit, you are required to uninstall the previous version within 120 days of installing your newer product. See your license agreement for more information.

Preparing a Deployment

To prepare for a deployment, you should take the time to review the following requirements and options. In most cases, you may install using the default settings. The language and default units are based on the country you selected in the License Agreement dialog. After install, you can use content and units from different regions by selecting them from the Content Selection screen.

Configure Individual Products

You can create a deployment that uses the default product settings, or you can configure each product to be included in the deployment. To configure a product, on the Create Deployment page of the deployment wizard, select the appropriate product from the drop-down list, and click Configure.
